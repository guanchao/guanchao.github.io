## 欢迎！👋

这是我使用 Hugo + Stack 主题搭建的个人博客。

### 为什么写博客？

- 📝 记录学习过程
- 💡 分享知识和经验
- 🚀 持续成长

### 技术栈

这个博客使用了：

- **Hugo** - 快速的静态网站生成器
- **Stack主题** - 现代、简洁的博客主题
- **GitHub Pages** - 免费的托管服务
- **GitHub Actions** - 自动化部署

### 开始写作

只需要在 `content/post/` 目录下创建新的 markdown 文件，然后：

```bash
hugo new post/my-new-post/index.md
```

编辑完成后，提交到 GitHub，GitHub Actions 会自动构建和部署！
